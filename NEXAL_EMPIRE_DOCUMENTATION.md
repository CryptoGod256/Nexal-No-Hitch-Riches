### 3. Subsystem Breakdown

#### A. NEXAL_BRIDGE Engine (`portal.html`)
- **Resilient Polling:** Automated 5000ms fetch loop (`/api/signals.json`) with exponential backoff on connection fault (up to 15,000ms).
- **Event Dispatcher:** Dispatches custom `nexaSignal` events to synchronize state across UI components in real time.

#### B. Video & Live Streaming Hub (`streaming/index.html`)
- **Twitch x YouTube Hybrid Architecture:** Combines Twitch's real-time live telemetry sidebar and command terminal with YouTube's clean workflow discovery grid.
- **Active AI Agent Telemetry:** Live monitoring for OpenClaw, Autonomous Engine, and Social Hub Sync.
- **B2B Actions:** Enterprise API tipping in $NEXA, one-click workflow cloning, and raw log exports.
- **Short-Form Execution Reels:** Instant visual clips for SQLite ingestion, peer handshakes, and bridge fault-tolerance routines.

#### C. Open Claw Design Ingestion (`open_claw_designer.py`)
- Automated extraction of structural blueprints and UI patterns to populate interactive sub-portals dynamically.
